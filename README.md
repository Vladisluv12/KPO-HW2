# ВШЭ Банк- система управления аккаунтами, категориями и операциями
## Технологии
- .NET 8.0
- Spectre.Console (UI) (запускалось в консоли Rider на Windows, если возникнут проблемы с кодировкой)
- Microsoft.Extensions.DependencyInjection для Dependency Injection из коробки
- YamlDotNet для импорта файлов из .yaml

## Архитектура
Нет обширного дерева наследования. Из предложенной опциональной функциональности реализованы: аналитика, импорт и экспорт данных, статистика.

## Паттерны GoF
1. Singleton - можно считать класс DIContainer синглтоном, реализован в качестве статического класса.
2. Abstract factory - есть одна фабрика, которая создаёт разные объекты доменной модели.
3. Facade - для взаимодействия с объектами используются фасады.
4. Command - также используются команды для обертки функций фасадов.
5. Decorator - используется для обёртки команд и измерения времени выполнения команды.
6. Template method - импорт данных с помощью класса DataImporter.
7. Visitor - экспорт данных, реализован с aбстрактным классом File и интерфейсом IVisitor.
8. Proxy - реализованы кешированные репозитории, но они не используются в данной версии.

## Принципы SOLID, GRASP
**SRP** - каждый класс имеет свою задачу.

**OCP** - легко расширить уже существующие классы.

**LSP** - наследники заменяют родителей.

**ISP** - интерфейсы реализуют только необходимые методы и свойства.

**DIP** - зависимости прокидываются через DIContainer.

**Low coupling** - нет сильно связанных классов.

**High cohesion** - классы спроектированы под конкретные задачи.

## Как запустить
После скачивания проекта и захода в директорию:
```
dotnet restore
dotnet build
dotnet run --project FinanceAccounting
```
